# CLAUDE.md — GLAMRDiP Operations Vault

## What This Is
This is the operational knowledge base for GLAMRDiP, a nail dip powder brand sold in 4 regions (AUS, UK, CA, Nordic). Scale Labs (an Australian agency) manages inventory and ops. This vault contains everything needed to run inventory operations and (eventually) marketing.

## Vault Map
- `Ops/` — inventory, fulfilment, supply chain, 3PLs, ordering, stock analysis
- `Marketing/` — campaigns, CX comms, launches, ads, socials (being built out)
- `Shared/` — products, people, component maps (used by both domains)
- `Archive/` — completed reports, old weekly summaries

## Routing — Which Domain?
- Inventory, stock, orders, suppliers, 3PL, fills, containers, reviews, recaps, POS model, sales analysis → Read `Ops/CLAUDE.md` then proceed
- Campaigns, emails, launches, CX, customer comms, socials, colour launches → Read `Marketing/CLAUDE.md` then proceed
- If the task touches both (e.g. "which colours are OOS that we're about to launch?") → read both domain CLAUDE.md files
